# TravelGlobe

A personal travel journal & visualization platform with 3D globe and AI-assisted travel features.

## Features
- Interactive 3D Earth & Mood Sphere (Three.js / R3F)
- AI-generated travel diary
- AI travel planning assistant
- Google Maps integration
- Photo upload & gallery

## Tech Stack
- React 18 + TypeScript
- Three.js / @react-three/fiber
- Tailwind CSS
- Zustand
- Gemini API
- Google Maps API
- Cloudinary

## Highlights
- Custom 3D shader & instanced geometry
- Performance-optimized scene switching
- Responsive 3D interaction design
- AI feature integration without backend model training

## Performance & Compatibility
- Desktop: Fully featured
- Mobile: Optimized
