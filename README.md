# Travel Globe - Prototype (TypeScript)

Minimal Vite + React + @react-three/fiber + drei + Tailwind + i18next prototype.

## Node
Recommended Node: 20.19.0

## How to run
1. npm install
2. npm run dev
3. Open the local url shown by Vite (e.g. http://localhost:5173)

## Features
- 左侧 Sidebar（语言切换 / 用户 / 添加按钮）
- 中央 3D Canvas（地球 / 心情球 切换）
- 右侧信息卡（最新日记 / 下一站愿景）
